# Portfilio_Dev
This document provides comprehensive technical documentation for the Portfolio Website project developed by Clement Wong Qin Yuan. The application is a modern, responsive web portfolio built using React 19.1.0 and Vite 7.0.3, designed to showcase professional projects, technical skills, and career experience in an interactive and visually appealing format.

-------------------------------------------------------------------------------------------------------------------------------------------------
# Installation & Setup
1. Repository Setup
bash# Clone the repository
git clone https://github.com/Clement0803/Portfilio_Dev.git

cd Portfilio_Dev/portfolio

# Install dependencies
npm install

npm run dev

# Access application

http://localhost:5173

# Component Architecture

The application is built as a single main component with multiple sections:

Navigation Component: Fixed header with smooth scrolling

Hero Section: Personal introduction and contact information

About Section: Professional background and interests

Skills Section: Technical competencies display

Projects Section: Featured work showcase

Experience Section: Professional timeline

Contact Section: Communication channels

Footer Component: Copyright and additional information
